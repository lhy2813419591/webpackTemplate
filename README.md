## webpack4多页应用框架

#### 使用方式

`npm install`

先安装依赖(有yarn也可以使用yarn进行安装)



## 项目文件如下图

![1561535151693](C:\Users\LJC\AppData\Roaming\Typora\typora-user-images\1561535151693.png)

- src文件夹
  - 其中的page文件夹存放html页面代码
  - js文件夹存放js文件,作为webpack打包的入口文件,在js中引入css以及js插件(需与html同名)



## 打包代码

运行`npm run prod`

即可打包项目代码
